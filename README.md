# CA_LAUNCHER

A Hub to facilitate teacher's access to some websites.

## Features

*   Provides a central location for teachers to access frequently used websites.
*   Includes links to Aula Digital, Blinklearning, CambridgeGo, Escola Virtual, Google, Microsoft, Teams, and YouTube.
*   Simple and easy-to-use interface.

## Usage

Teachers can use this launcher to quickly access the resources they need for their daily tasks. Simply open the `index.html` file in a web browser.
